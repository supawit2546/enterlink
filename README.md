# Enterlink — Link in Bio for Business

A Linktree-inspired platform built for businesses and entrepreneurs 
in Thailand. Helps customers find official channels easily and 
reduces risks from fake pages or phishing links.

🔗 [Live Demo](https://enterlink.vercel.app/)

## Features
- 🖼️ Profile image upload
- 🔗 Multi-platform link management
- 🎨 Customizable components and themes
- ✏️ Editable personal information

## Tech Stack
**Frontend:** Next.js, React.js, Tailwind CSS  
**Backend:** Golang, GORM  
**Database:** PostgreSQL  
**Deployment:** Vercel
